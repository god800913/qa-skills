## 버그 리포트

**제목:** `[라운지 추천] 딥링크 진입 시 스크롤 최상단 고정 — iOS 17.5 앱 ver118`

**환경:** iPhone 14 / iOS 17.5 / 앱 ver118

**사전 조건:** 로그인 상태, `enableLoungeRecommend` ON, 라운지 추천 섹션 노출 중

**재현 단계:**
1. 홈 화면에서 라운지 추천 섹션의 특정 카드를 탭 (딥링크 진입)
2. 라운지 상세 화면 진입 확인
3. 상세 화면 내 스크롤 위치 확인
4. 뒤로가기 후 다시 같은 카드 탭 → 재진입

**기대 결과:** 딥링크로 진입한 라운지 상세 화면이 해당 콘텐츠 위치로 스크롤 이동

**실제 결과:** 매 진입 시 스크롤이 최상단으로 고정됨 — 대상 콘텐츠가 화면 밖으로 밀려남 (재진입 포함 5/5 재현)

**심각도:** Major  
**근거:** 핵심 CTA(추천 콘텐츠 탐색)에 인접하나, 수동 스크롤로 우회 가능 → Blocker 기준 미충족. 추천 섹션 핵심 플로우이므로 Minor 상향하여 Major 판정.

**재현 빈도:** 5회 중 5회

**관련 TC_ID:** `1-4`

**첨부:** `lounge_deeplink_scroll_ver118.mp4` (스크린 레코딩)
